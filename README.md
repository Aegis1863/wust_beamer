## 内容说明

- ref.bib 整理参考文献，在引用论文时把bib格式的cite放在这里
- slide.tex 幻灯片tex文件，在这里编辑幻灯片主要内容
- WUST.sty 武汉科技大学主题配置文件
- pic文件夹 用于存放图片
- 其他文件为编译中间文件，非必要
